# charCreator
 creates character for writting and dming purposes momentally

## Install of repository
 1) install nodeJS (I personally use v24.18.0)  link here: https://nodejs.org/en/download
 2) if not done already clone repository from github
 3) go to cmd, run npm install on project path
 4) develop given task and then commit

## dev testing
 - npm start => in cmd (run without building final product to test the feature)
 - npm make => in cmd (to generate exe file locally and try to run it)
